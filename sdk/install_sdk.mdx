Before installing the SDK, ensure you have:

- **An account** – You will need an API key and merchant credentials
- **A supported programming environment** – (e.g., Node.js, JavaScript)

# Installing AcountPay SDK

AcountPay SDK can be installed via npm or yarn.

## Prerequisites

- Node.js 14.x or higher
- React 16.8+ (for React implementation)
- A modern browser supporting ES6

## Installation

Install the SDK using **npm**:

```javascript javascript
npm i @acountpay-sdk
```

Install the SDK using **yarn**:

```javascript
yarn add @acountpay-sdk
```
