Before installing the SDK, ensure you have:

* **An account** – You will need an API key and merchant credentials
* **A supported programming environment** – (e.g., Node.js, JavaScript)

## Installation

Install the SDK using **npm**:

```javascript javascript
npm install acountpay-sdk

```
Install the SDK using **yarn**:

```javascript
yarn add acountpay-sdk
```
