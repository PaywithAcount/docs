Before making any method calls, initialize the SDK with your Client ID and configure the API base URL at runtime.

## Constructor

Create an instance with your `clientId`. You can optionally pass `apiBaseUrl`, `authToken`, and other options.

```ts
import AcountPay from '@acountpay/sdk';

const sdk = new AcountPay({
  clientId: process.env.REACT_APP_ACOUNTPAY_CLIENT_ID!,
  // Option A: pass base URL here
  // apiBaseUrl: '[https://acount-apis-7886a9e16dc3.herokuapp.com](https://acount-apis-7886a9e16dc3.herokuapp.com)',
  // Optional (if backend requires auth):
  // authToken: '<Bearer token>',
  // extraHeaders: { 'X-Trace-Id': '...' },
});

Supported options:

clientId: string (required)
apiBaseUrl?: string (recommended; see Runtime Config)
env?: string (optional)
redirectUri?: string (optional)
authToken?: string (optional; sends Authorization: Bearer )
extraHeaders?: Record<string, string> (optional; appended to requests)

## Runtime configuration (recommended)
Set the API base URL at runtime to avoid rebuilds.

```html
<!-- public/index.html -->
<script>
  window.__ACOUNTPAY_API_URL__ = 'https://acount-apis-7886a9e16dc3.herokuapp.com';
  // Optional: provide bearer token if your backend requires auth
  // window.__ACOUNTPAY_AUTH_TOKEN__ = '<Bearer token>';
</script>
```

The SDK will automatically use window.__ACOUNTPAY_API_URL__ when present. You can still override it via the constructor apiBaseUrl.

Environment variables (React example)

```bash
# .env
REACT_APP_ACOUNTPAY_CLIENT_ID=your-client-id
REACT_APP_ACOUNTPAY_REDIRECT_URI=https://www.acountpay.com//payment-callback
# Optional
REACT_APP_ACOUNTPAY_ENV=sandbox
```

Usage in React:
```typescript
import AcountPay from '@acountpay/sdk';

const sdk = new AcountPay({
  clientId: process.env.REACT_APP_ACOUNTPAY_CLIENT_ID!,
});
```

## Best practices
Use major currency units for amount (e.g., 1 = 1 DKK).
Ensure your app origin and callbackURL are whitelisted on your backend.
Provide a bearer token at runtime if your environment requires auth (do not hardcode secrets).
Prefer runtime base URL configuration to switch environments without rebuilding.
