# Settings & Configuration

Customize your AcountPay POS system to match your business needs with comprehensive settings for payments, tips, business information, and more.

## Tip Management

### Tip Configuration

<img src="/images/tip-settings.png" alt="Tip Settings" style={{borderRadius: '8px', border: '1px solid #e1e5e9'}} />

**Global Tip Settings:**
- **Enable/Disable Tips**: Toggle tip functionality system-wide
- **Default Percentages**: Set standard tip options (10%, 15%, 20%, 25%)
- **Custom Amount**: Allow customers to enter custom tip amounts
- **Mandatory Tips**: Make tips required for certain transaction types

**Tip Configuration:**
```javascript
const tipSettings = {
  enabled: true,
  defaultPercentages: [10, 15, 20, 25],
  allowCustomAmount: true,
  mandatoryForServices: false,
  displayBehavior: {
    showOnAllTransactions: true,
    skipOption: true,
    customerFacing: true
  },
  minimumTipAmount: 0.50,
  maximumTipPercentage: 50
};
```

### Tip Display Options

**Customer Interface:**
- **Customer-facing Display**: Tips shown to customer during payment
- **Merchant Control**: Staff can add tips on customer's behalf
- **Receipt Options**: Show tip breakdown on receipts
- **Digital Receipts**: Include tip in email/SMS receipts

**Advanced Tip Features:**
- **Service-based Tips**: Different tip settings for different services
- **Time-based Tips**: Automatic tip suggestions based on time of day
- **Employee Tip Tracking**: Monitor tip distribution to staff
- **Tip Pool Management**: Distribute tips among team members

## Payment Settings

### Currency Configuration

<img src="/images/payment-settings.png" alt="Payment Settings" style={{borderRadius: '8px', border: '1px solid #e1e5e9'}} />

**Supported Currencies:**
- **Primary Currency**: Main business currency (EUR, DKK, SEK, NOK, GBP)
- **Multi-currency Support**: Accept payments in multiple currencies
- **Exchange Rates**: Automatic or manual rate updates
- **Display Format**: Currency symbol position and decimal places

**Currency Settings:**
```javascript
const currencyConfig = {
  primary: "EUR",
  supported: ["EUR", "DKK", "SEK", "NOK", "GBP"],
  displayFormat: {
    symbol: "â‚¬",

    decimals: 2,

  exchangeRates: {
    autoUpdate: true,

    updateFrequency: "daily"
```

### Payment Processing

**Default Settings:**
- **Payment Methods**: Enabled payment types
- **Transaction Limits**: Minimum and maximum amounts
- **Timeout Settings**: Payment expiration times
- **Retry Logic**: Failed payment handling

**Processing Configuration:**
```javascript
const paymentConfig = {
  methods: {
    bankTransfer: {
      enabled: true,
      minAmount: 0.01,
      maxAmount: 10000.00,
      timeout: 300 // seconds
    },
    qrCode: {
      enabled: true,
      expirationTime: 600, // seconds
      refreshable: true
    }
  },
  retryPolicy: {
    maxAttempts: 3,
    backoffStrategy: "exponential",
    timeoutIncrement: 30
  }
};
```

```javascript

  defaultRate: 25.0, // 25% VAT
  rates: {
    standard: 25.0,
    reduced: 12.0, // for food items

    books: "books",
    services: "standard"

  reporting: {
    frequency: "monthly",

};
```

## Security Settings

### Access Control

<img src="/images/security-settings.png" alt="Security Settings" style={{borderRadius: '8px', border: '1px solid #e1e5e9'}} />

**Authentication:**
- **Two-factor Authentication**: Enhanced security for admin accounts
- **Password Requirements**: Complexity and expiration policies
- **Session Management**: Automatic logout and session timeouts
- **IP Restrictions**: Limit access to specific IP addresses

**Security Configuration:**
```javascript
const securityConfig = {
  authentication: {
    twoFactorRequired: true,
    passwordPolicy: {
      minLength: 8,
      requireUppercase: true,
      requireNumbers: true,
      requireSymbols: true,
      expirationDays: 90
    },
    sessionTimeout: 30, // minutes
    maxFailedAttempts: 5
  },
  access: {
    ipWhitelist: ["192.168.1.0/24"],
    allowedCountries: ["DK", "SE", "NO"],
    restrictedHours: {
      enabled: false,
      start: "22:00",
      end: "06:00"
    }
  }
};
```
