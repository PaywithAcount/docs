### Creating Products

**Product Information:**
- **Name**: Product title and description
- **SKU**: Unique product identifier
- **Price**: Base price with currency support
- **Category**: Organizational grouping
- **Stock Level**: Inventory tracking


## Cart Functionality
- Add products with quantity selection
- Modify quantities after adding
- Remove individual items
- Clear entire cart
- Save cart for later completion

## Bulk Product Import

**Step 1: Prepare CSV File**

Required CSV format:
```csv
name,description,sku,price,category,stock_level,image_url
Premium Coffee Beans,Single origin arabica,COFFEE-001,24.99,beverages,150,https://example.com/coffee.jpg
Ceramic Mug,Handcrafted ceramic mug,MUG-001,12.50,accessories,75,https://example.com/mug.jpg
Croissant,Fresh baked croissant,FOOD-001,3.50,food,0,https://example.com/croissant.jpg
```

**CSV Field Descriptions:**
- `name`: Product name (required)
- `description`: Product description
- `sku`: Unique identifier (required)
- `price`: Product price (required)
- `category`: Product category
- `stock_level`: Initial inventory level
- `image_url`: Product image URL

**Step 2: Upload and Validation**

**Import Process:**
1. **File Upload**: Drag-and-drop or browse for CSV file
2. **Data Parsing**: Automatic column mapping and validation
3. **Error Detection**: Identify and highlight data issues
4. **Preview**: Review parsed data before import
5. **Confirmation**: Execute the import process

**Validation Checks:**
- Required fields completion
- Price format validation
- SKU uniqueness verification
- Category existence check
- Image URL accessibility
