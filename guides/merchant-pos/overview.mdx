# AcountPay Merchant POS Platform

AcountPay is a comprehensive Point of Sale (POS) platform designed for merchants to accept bank account payments via QR codes and invoices. The platform integrates with Token.io for secure payment processing and provides a modern, touch-friendly interface optimized for mobile devices and tablets.

- **Bank-to-Bank Payments**: Direct transfers without card networks
- **QR Code Integration**: Modern payment method customers expect
- **Real-time Processing**: Instant payment confirmation

## How It Works

The AcountPay POS platform follows a simple, secure payment flow:

1. **Transaction Setup**: Merchant enters payment details and asks customer for their bank
2. **Bank Selection**: Choose customer's bank from supported institutions list
3. **QR Code Generation**: System generates secure QR code for selected bank
4. **Customer Scanning**: Customer scans QR code with their banking app
5. **Secure Processing**: Payment processed through Token.io infrastructure
6. **Real-time Confirmation**: Instant notification of payment completion
